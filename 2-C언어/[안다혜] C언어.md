C로 작성한 코드는 "파일이름.c"로 저장한다. (C로 작성된 코드라는 의미)

우리가 작성한 `소스 코드`를 2진수로 변환하는 작업을 `컴파일러`가 해준다. 컴퓨터가 이해할 수 있도록.

```
./a.out # 현재 디렉토리에 있는 a.out이란 프로그램을 실행하게 해줌.
```
맨 앞의 .은 현재 폴더를 나타낸다.
