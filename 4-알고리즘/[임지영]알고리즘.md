# Topic 1 알고리즘

전화번호부 책에서 이름을 찾을 때
1장씩 넘기는 알고리즘? -> 효율적이지 않다.
2장~3장씩 넘기는 알고리즘? -> 빠르지만 원하는 이름을 못 찾을 수 있다.
중간 이상부터 보는 알고리즘 -> 찾을 것들이 반으로 준다.
계속 1/2 해서 책을 반으로 자른 다음에 한쪽을 버리고 한쪽을 찾는 알고리즘 ---> 빠르고 정확하다
