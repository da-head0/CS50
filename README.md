## 깃허브 잔디 채우면서 스터디하기

- 이 깃허브 폴더에 개인별로 .md 파일을 생성, 그날 공부한 내용을 정리하고 저장
- ex) `1-컴퓨터 구조/[김코딩]컴퓨터구조.md`
- 마크다운 파일 생성 시 잔디가 한 번 심어지며, 수정할 때마다 잔디가 심어집니다. 각 챕터마다 한 파일에 꾸준히 정리하시면 됩니다!
